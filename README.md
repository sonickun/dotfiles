# dotfiles

zsh / vim / tmux の設定ファイル群です。

## 前提条件

以下のツールがインストールされていること。

- **zsh** — シェル本体
- **oh-my-zsh** — zsh プラグイン管理フレームワーク
- **vim** — テキストエディタ
- **tmux** — ターミナルマルチプレクサ

## セットアップ

```bash
git clone <this-repo> ~/dotfiles
cd ~/dotfiles
bash setup.sh
```

`setup.sh` が行うこと:

- `.vimrc`, `.tmux.conf` のシンボリックリンクを `~` に作成
- NeoBundle (vim プラグインマネージャ) をインストール
- カラースキーム `hybrid` をインストール
- oh-my-zsh をインストールし、`.zshrc` のシンボリックリンクを作成
- デフォルトシェルを zsh に変更

## 構成ファイル

| ファイル | 説明 |
|---|---|
| `.vimrc` | Vim 設定 (NeoBundle, hybrid カラースキーム, lightline) |
| `.zshrc` | Zsh 設定 (oh-my-zsh, blinks テーマ) |
| `.tmux.conf` | tmux 設定 (プレフィックス C-a, vi キーバインド, マウス操作) |
| `blinks.zsh-theme` | カスタム Zsh テーマ (Solarized 対応) |
| `setup.sh` | セットアップスクリプト |

## tmux 主要コマンド

プレフィックスキーは `C-a`（Ctrl + a）です。

### セッション

| コマンド | 説明 |
|---|---|
| `tmux new -s <name>` | セッションを名前付きで作成 |
| `tmux ls` | セッション一覧を表示 |
| `tmux attach -t <name>` | セッションにアタッチ |
| `C-a d` | セッションをデタッチ |
| `C-a $` | セッション名を変更 |

### ウィンドウ

| コマンド | 説明 |
|---|---|
| `C-a c` | 新しいウィンドウを作成 |
| `C-a n` | 次のウィンドウへ移動 |
| `C-a p` | 前のウィンドウへ移動 |
| `C-a <number>` | 番号指定でウィンドウへ移動 |
| `C-a ,` | ウィンドウ名を変更 |
| `C-a &` | ウィンドウを閉じる |
| `C-h` / `C-l` | 前 / 次のウィンドウへ移動 (リピート可) |

### ペイン

| コマンド | 説明 |
|---|---|
| `C-a \|` | 左右に分割 |
| `C-a -` | 上下に分割 |
| `C-a h/j/k/l` | 左 / 下 / 上 / 右のペインへ移動 |
| `C-a H/J/K/L` | ペインをリサイズ (リピート可) |
| `C-a x` | ペインを閉じる |
| `C-a z` | ペインをズーム (全画面表示切替) |
| `C-a {` / `C-a }` | ペインを入れ替え |

### コピーモード (vi キーバインド)

| コマンド | 説明 |
|---|---|
| `C-a [` | コピーモードに入る |
| `v` | 選択開始 |
| `y` | ヤンク (コピー) |
| `q` / `Esc` | コピーモードを抜ける |
| `C-a ]` | ペースト |

### その他

| コマンド | 説明 |
|---|---|
| `C-a r` | 設定ファイルをリロード |
| `C-a :` | tmux コマンドプロンプトを開く |
| `C-a ?` | キーバインド一覧を表示 |
| マウスホイール | スクロール (マウス操作有効) |
